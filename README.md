# Student Performance EDA

Exploratory Data Analysis датасета успеваемости португальских студентов по математике.

## Датасет
- **Источник:** UCI Machine Learning Repository
- **395 студентов**, 33 признака
- Данные собраны в двух португальских школах

## Цель
Найти факторы влияющие на итоговую оценку G3 (шкала 0-20).

## Ключевые выводы
- Главный предиктор G3 — предыдущие оценки G1 и G2 (корреляция ~0.80-0.90)
- Неудачи (failures) сильно снижают итоговую оценку
- Оптимальное время учёбы — 5-10 часов в неделю
- Алкоголь слабо но отрицательно влияет на успеваемость
- ~10% студентов получили 0 — вероятно бросили курс

## Стек
Python · Pandas · Seaborn · Matplotlib
